# tech-challenge-fase1-saude-mulher

Sistema de IA focado em Machine Learning, permitindo que dados médicos sejam analisados automaticamente para identificar padrões de risco relacionados à segurança e saúde das mulheres.

### 1. Criar e Ativar Ambiente Virtual

**Windows:**

```bash
python -m venv venv
venv\Scripts\Activate.ps1
```

### 2. Instalar Dependências

Utilize o comando abaixo para instalar as bibliotecas necessárias:

```bash
pip install -r requirements.txt
```

### 3. Desativar Ambiente Virtual

deactivate
